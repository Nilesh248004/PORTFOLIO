# Nilesh Krishnamurthy Portfolio

Responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Features

- Dark responsive portfolio design
- Mobile-friendly navigation
- Dynamic skill and project counters
- EmailJS contact form integration
- SEO metadata, manifest, robots.txt, and sitemap.xml

## Setup

Update the EmailJS configuration in `script.js`:

```js
const emailJsConfig = {
  serviceId: "YOUR_SERVICE_ID",
  templateId: "YOUR_TEMPLATE_ID",
  publicKey: "YOUR_PUBLIC_KEY",
};
```

Update `sitemap.xml` and `robots.txt` with your deployed domain.

## Run Locally

Open `index.html` directly in your browser.
